# Installation

## Requirements

- Python 3.10+

## Core Installation

```bash
pip install ddacs
```

## Optional Dependencies

### PyTorch Integration

```bash
pip install "ddacs[torch]"
```

### Examples (with visualization)

```bash
pip install "ddacs[examples]"
```

### Development

```bash
git clone https://github.com/BaumSebastian/DDACS.git
cd DDACS
pip install -e ".[dev]"
```
